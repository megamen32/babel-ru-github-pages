README — Babel RU

Бесконечная библиотека Борхеса на JavaScript.
Не генератор текста. Не ИИ-чат. Не база данных.

Это математически адресуемая библиотека, где каждая возможная страница уже существует.

✨ Что это вообще такое

Babel RU — русскоязычная reinterpretation идеи The Library of Babel.

Каждая страница:

* имеет фиксированную длину 4096 символов;
* состоит из алфавита 256 символов;
* существует всегда;
* вычисляется из координат;
* не хранится на сервере.

То есть библиотека не “генерирует” текст.
Она вычисляет, какая книга лежит по этому адресу.

⸻

🌌 Главная идея

Ты не ищешь текст.

Ты блуждаешь по бесконечному пространству координат:

зал → стена → полка → том → страница

И где-то там уже существует:

* твоя биография;
* рецепт борща;
* расшифровка инопланетян;
* идеальный роман;
* все ошибки;
* все возможные диалоги;
* случайный шум.

🧠 Архитектура

Алфавит

Библиотека использует собственный 256-символьный алфавит:

* русский;
* английский;
* цифры;
* знаки;
* emoji.

Каждый символ = ровно 1 байт.

⸻

Размер страницы

4096 символов
4096 × 8 = 32768 бит

Каждая страница — это число размером 32768 бит.

⸻

Перестановка

Используется обратимая аффинная перестановка:

y = C·x + OFFSET mod 2^32768

Где:

* x — индекс страницы;
* y — содержимое;
* C — широкая avalanche-константа;
* OFFSET — глобальный сдвиг.

⸻

⚠️ Важный фикс avalanche effect

Ранний прототип имел критическую проблему:

PERM_C ≈ 2^63
modulus = 2^32768

Из-за этого соседние страницы совпадали на 99.8%.

Исправление:

* PERM_C теперь имеет биты по всей ширине 32768;
* изменение индекса меняет ВСЮ страницу;
* соседние страницы имеют ~0% сходства.

Тесты находятся в:

tests/test_adjacent_similarity.js

🎨 Почему проект выглядит так

Большинство “библиотек Борхеса” выглядят как:

* случайный текст;
* скучная сетка;
* криптографическая демка.

Здесь идея другая:

библиотека должна ощущаться как место.

Не “генератор”.
Не “алгоритм”.

А пространство, где хочется блуждать.
