# Русская Библиотека Вавилона — реальная алгоритмическая версия

Это версия без хранения страниц, без индекса и без payload с фразой.

## Принцип

```text
страница → BigInt → base36-адрес
base36-адрес → BigInt → страница
```

Адрес является числом всей страницы. Страница восстанавливается из адреса.

## Что улучшено

- красивый вавилонский адрес с дефисами;
- несколько вариантов страницы для одной фразы;
- выбор положения фразы: начало, центр, конец, разные места, тихое окружение;
- две версии алгоритма:
  - `v1` — 900 символов;
  - `v1s` — короткая версия 300 символов;
- проверка обратимости на странице;
- локальное избранное через `localStorage`;
- скачивание текста страницы;
- скачивание карточки страницы как PNG;
- страница «Как работает»;
- GitHub Pages workflow.

## Формат ссылки

```text
#/v1/a/<base36-address-with-dashes>
```

Например:

```text
#/v1/a/abc12345-def67890-...
```

## Важное правило

Нельзя менять алгоритм `v1` после публикации:

- алфавит;
- длину страницы;
- нормализацию текста;
- порядок кодирования.

Если нужно изменить алгоритм — добавляй новую версию: `v2`.

## Локальный запуск

```bash
python3 -m http.server 8000
```

Открыть:

```text
http://localhost:8000/
```

## GitHub Pages


В GitHub включить:

```text

```
